Month in Review // December 2015

Published // 5 January, 2016  |  Column // Month in Review

So, this has been a pretty big month for me. Not only in terms of the festive season, but also as I've relaunched my illustration and design business. I've been relearning how to blog and promote my work, hopefully in a more successful way than before.

So what have I been up to over the month of December, not just in terms of work, but fun as well?

Other Happenings This Month

  • Reading // It by Stephen King. Well, I’ve been reading it since August. It is so VERY long. Parts I’ve loved, other parts I was just feeling desperate to get to the end as my interest (and the story) waxed and waned (Follow me on Goodreads)
  • Watching // The Good Wife. All kinds of love for this show. I’m into season five now, but Netflix doesn’t have past this point! Please, Netflix, can we have the rest?
  • Listening // Weezer, Thank God for Girls
  • Playing // The Legend of Zelda: Ocarina of Time. I’m so bad at this kind of game. I can never figure out what I’m supposed to do! But then I’ve rarely played this type of game before, so maybe I need to train myself more?
  • Eating // ALL the Christmas goodness! Especially stuffing, ham and sprouts
  • Learning // Tying to learn how to run my new business! Baby steps …
  • Doing // Finishing off painting our flat. Some more trim work do do ;_;
